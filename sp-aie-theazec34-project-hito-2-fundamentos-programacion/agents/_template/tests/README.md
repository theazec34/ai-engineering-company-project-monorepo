# `agents/_template/tests` folder

This folder defines the **test skeleton** for AI agents created in the monorepo from the `agents/_template/` template.

You should include examples of functional, regression, and quality tests here (for example: input prompts, expected responses, edge cases, automated validations, etc.) that can be reused or adapted when building new agents.

- **Main purpose**: provide a **minimal reference for testing best practices** for all company agents in the monorepo.
- **Recommendation**: document the test types, tools, and acceptance criteria you use, and extend this pattern when new agents or use cases are added.
