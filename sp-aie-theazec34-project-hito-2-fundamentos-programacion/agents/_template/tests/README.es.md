# Carpeta `agents/_template/tests`

Esta carpeta define el **esqueleto de pruebas** para los agentes de IA que se creen en el monorepo a partir de la plantilla de `agents/_template/`.

Aquí deberías incluir ejemplos de tests funcionales, de regresión y de calidad (por ejemplo: prompts de entrada, respuestas esperadas, escenarios límite, validaciones automáticas, etc.) que se puedan reutilizar o adaptar al crear nuevos agentes.

- **Propósito principal**: ofrecer una **referencia mínima de buenas prácticas de testing** para todos los agentes de la compañía dentro del monorepo.
- **Recomendación**: documenta los tipos de pruebas, herramientas y criterios de aceptación que uses, e invita a extender este esquema cuando se añadan nuevos agentes o casos de uso.
