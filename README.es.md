## Comandos

- `npm install`
- `npm run typecheck` — validar TypeScript (`tsc --noEmit`)
- `npm run demo` — demo por consola
- `npm run build:web` y `npm run serve` — prueba en navegador en `http://localhost:3000/src/index.html`
